# SONA 2026 (Static Site)

Simple static site scaffold using plain HTML, CSS, and JavaScript.

## Structure
- `index.html`, `candidates.html`, `issues.html`, `contact.html`
- `components/` — `navbar.html`, `footer.html`, `hero.html`
- `styles/main.css`
- `scripts/main.js`
- `assets/` — images like `hero.svg`

## Local
Open `index.html` in a browser (or use a local static server / Live Server extension).

## Deploy to Vercel
1. Create a Git repo and push this project.
2. Import the repo on Vercel. No build step required.

That's it — the site is ready for content and small iterations.
